How to use Hexo to build your blog on Github

This document will teach you how to build your blog on github based on the hexo framework(which is a static blog framework).

Quick Start

Step 1

Create a new repository in github, and the repository name must be your github usename with the suffix of “.github.io”.
For example, my username is shizihao123, the new repository is shizihao123.github.io. Open https://username.github.io and you will see your blog!!!

Step 2

Install Hexo

Step 3

Set up and Configure Hexo
Key point : Open _config.yml, set your url to https://username.github.io and set root as /.

Step 4

Edit your blog according to the hexo guide.

Step 5

Use the basic command to render, generate and deploy your blog.

Cong! You have finished the building of your blog with hexo, and write what you want now !
